ABOUT

ステージの外で始まる、もうひとつの物語。

『Offstage Love』は、芸能界に飛び込んだあなたが、トップアイドルグループ・ラグティアとの仕事と出会いを通じて、夢と恋の三年間を歩む女性向けキャラクターIP型ライフシミュレーションです。

Work × Love

仕事と恋を、同じ時間で進める。

芸能活動、自己成長、休日行動、出会い。恋だけではなく、仕事を通じて関係が積み上がっていきます。

Common Route

誰と恋に落ちても正解。

本編は共通ルート制。最初から一人に固定せず、三年間の積み重ねの先で向き合う相手を選びます。

Music Memory

想い出は、音で残る。

ストーリーや寄り道の記憶が楽曲として解放され、あとから振り返ることができます。
