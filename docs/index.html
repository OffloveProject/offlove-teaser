ABOUT

ステージの外で始まる、もうひとつの物語。

『Offstage Love』は、芸能界で出会う、仕事と恋と音楽の物語。仕事と恋愛を同じ時間で積み上げる体験を目指しています。

Work × Love

仕事と恋を、同じ時間で進める。

芸能活動、自己成長、休日行動、出会い。恋だけではなく、仕事を通じて関係が積み上がっていきます。

Common Route

三年間をどう積み上げるか。

プレイヤーは新人アーティストとして、日々の選択を重ねながら未来を作っていきます。

Music Memory

想い出は、音で残る。

ストーリーや寄り道の記憶が楽曲として解放され、あとから振り返ることができます。

GAME

あなたが主人公になるゲーム版。

ゲーム版では主人公名を固定せず、プレイヤー自身が新人アーティストとして芸能界での三年間を歩みます。
