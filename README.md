# Offstage Love teaser site

静的HTML/CSS/JSのみで構成したティザーサイト案です。
GitHub Pages等にそのまま配置できます。

## 方針

- `offstagelove.com` をIP全体の公式ホームとして扱う
- `/game/` はゲーム版。主人公名は出さない
- `/novel/` はノベル版。主人公名「橘 結月」はこのページでのみ使用
- `/characters/` は長期的な拡張を前提に、初期はラグティア5名を掲載
- 非掲載対象の将来構想・旧名称は公開ページに掲載しない
- noteへの導線は公開前のためdisabled状態

## 更新箇所

- noteマガジンURL: `assets/js/site-data.js` の `noteMagazineUrl`
- Google Analytics: `assets/js/analytics.js` の `OFFSTAGE_LOVE_GA_ID`
- OGP画像: `assets/img/ogp.png`
- ロゴ: `assets/img/offstage_love_logo.png`

## 注意

公開前に、全ページで旧名称や非掲載語が残っていないか確認してください。
